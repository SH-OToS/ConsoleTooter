# ConsoleTooter
## 説明
コンソールで作業中にちょっとTootしたくなったときに使えます      
一応TLも見れます(ガバガバ実装)     
## 書式
toot [内容]       
command | toot      
toot       
tlh      
tlh [リミット数]    
## 使い方
1. まず適当にパスが通っている場所にtootを置きます
2. お好きなインスタンスのアクセストークンを取得します
3. tootの$InstanceURLをhttps://[インスタンスのドメイン]/api/v1/statusesに書き換えます
4. tootの$TOKENをアクセストークンに書き換えます
5. コマンド感覚でTootできます
0. nullを草に置き換えると…
## タイムラインも見れるようにする場合
1. まず適当にパスが通っている場所にtlhを置きます
2. $InstanceURLをインスタンスのドメインに置き換える
3. $tokenをアクセストークンに置き換える
4. TLがコマンドで見れる
## 欠点
1. 通知が見れない
2. 私のやる気がない
3. シェルやcurlに突っ込むと怒られる内容は使えない
4. 結果が割りとがばい
5. タイムラインがカンマで改行して整形する仕様なので変なところで改行されることがある
6. タイムラインの表示を削るコードが本文を削ってしまう可能性がある
7. タイムラインの一部の文字がHTMLとJSONの境をうろうろする
8. タイムラインの絵文字がずれてる
9. プロフ画像が表示されない
10. タイムラインの時間表示が無い
11. タグ、画像、URLの含まれた投稿が整形されない
12. ブーストの表示がおかしい
13. CWが仕事しない
